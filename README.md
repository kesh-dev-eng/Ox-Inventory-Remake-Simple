# 🎒 Ox Inventory - Custom UI

A customized version of [ox_inventory](https://github.com/overextended/ox_inventory) with a unique dark theme and modified UI elements.

## 📸 Preview

![Inventory Preview](https://via.placeholder.com/800x400?text=Inventory+Screenshot)

## ✨ Features

- Custom dark theme with Oswald font
- Resized inventory slots (11.5vh)
- Hexagon-shaped control buttons
- Custom hotbar positioning
- Modified weight display styling
- Custom tooltip design

## 🚀 Installation

### Prerequisites
- FiveM Server
- [oxmysql](https://github.com/overextended/oxmysql)
